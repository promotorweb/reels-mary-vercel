# Mary Morgan • Tarot Reels

Projeto limpo em Vite + React + Tailwind, compatível com Vercel.

## Rodar localmente

```bash
npm install
npm run dev
```

## Deploy na Vercel

1. Suba esta pasta para um repositório no GitHub.
2. Importe o repositório na Vercel.
3. Framework: Vite.
4. Build command: `npm run build`.
5. Output directory: `dist`.

## Estrutura dos assets

- `public/assets/tarot/*.webp`
- `public/assets/mary/mary-morgan.webp`
